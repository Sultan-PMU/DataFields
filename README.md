# Data Fields 📊

## Overview

Data Fields is an educational website developed using HTML, CSS, and JavaScript. The project explains the differences between major data domains including Data Analysis, Data Science, Data Engineering, Data Management, and Business Intelligence.

The website helps students explore:

* Career paths
* Core tools and technologies
* Recommended certifications and courses
* Domain overviews and learning guidance

## Features

* Modern and responsive UI design
* Interactive pages using JavaScript
* Multiple educational sections for each data domain
* Simple navigation system
* Contact form page

## Technologies Used

* HTML5
* CSS3
* JavaScript

## Website Sections

* Home Page
* Data Analysis
* Data Science
* Data Engineering
* Data Management
* Business Intelligence
* About Page

## Project Goal

The main goal of this project is to help students understand the differences between data-related fields and guide them toward suitable learning paths and certifications.
